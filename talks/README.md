# 🎤 PyLadies Boston & Friday Noon Chats: Presentation Proposals

Welcome to the **Talks & Events** directory! This space compiles structured curriculum-alignment proposals designed for the **PyLadies Boston** chapter, highlighting both upcoming live events and Friday noon online chats tailored around community expertise.

This folder is organized as follows:
- [README.md](./README.md) (This file) — Folder index and overview.

## 💡 About These Proposals

These curriculum resources are generated dynamically by identifying key gaps between **local job postings** and **chapter event history** (leveraging live Google Jobs searches and Meetup feed data via SerpApi integrations).

By targeting cloud hosting (AWS/GCP), collaborative development, analytics engineering, and cutting-edge agentic systems, these sessions empower PyLadies members to acquire the exact skills that modern enterprise employers are searching for.

## 🗺️ 6-Week Talk Preparation Roadmap
**Constraint Checklist:**
- **Kickoff:** Next Week (Week 1)
- **Time commitment:** Max 5 hours per week
- **Scope:** 3 Talks (~10 hours of preparation per talk, split over 6 weeks, for a total of 30 hours)

### **Week 1: Kickoff & Structure Alignment (5 hours)**
*Goal: Finalize presentation structures and establish development environments.*
- **Task 1.1:** Review feedback on the three proposal reports and finalize any template modifications (1 hour).
- **Task 1.2:** Outline the slide deck structures (titles, talking points, and visual asset requirements) for all three talks (2 hours).
- **Task 1.3:** Verify and clean up the companion repositories on GitHub, ensuring they are public and the READMEs are clear (2 hours).

### **Week 2: Talk 1 (Model Context Protocol & FastMCP) Development (5 hours)**
*Goal: Build the slide deck and code demonstrations for the first talk.*
- **Task 2.1:** Draft the slides for Talk 1 focusing on the MCP client-server architecture (2 hours).
- **Task 2.2:** Build and dry-run the live coding portion of the talk, ensuring the FastMCP local server loads seamlessly (2 hours).
- **Task 2.3:** Record a quick 2-minute backup video of the live demo in case of network/live demo failures (1 hour).

### **Week 3: Talk 1 Review & Talk 2 (AI Agents) Design (5 hours)**
*Goal: Finalize Talk 1 with a dry run and begin preparing Talk 2.*
- **Task 3.1:** Perform a complete 45-minute timed dry run of Talk 1 and refine slide timings (1.5 hours).
- **Task 3.2:** Draft the slide deck for Talk 2, focusing on the conceptual differences between prompt engineering and autonomous agent patterns (ReAct) (2.5 hours).
- **Task 3.3:** Prepare the repository code walk-through for Talk 2 (1 hour).

### **Week 4: Talk 2 (AI Agents) Development & Talk 3 (Analytics Engineering) Design (5 hours)**
*Goal: Finalize Talk 2 and begin drafting the slides for Talk 3.*
- **Task 4.1:** Conduct a timed dry run of Talk 2 and refine the walk-through of the agent's decision loops (2 hours).
- **Task 4.2:** Draft the slide deck for Talk 3, focusing on mapping software engineering principles (Git, testing, schemas) to data pipelines (2.5 hours).
- **Task 4.3:** Choose the target dataset and query examples for the dbt/semantic layer demonstration (0.5 hours).

### **Week 5: Talk 3 (Analytics Engineering) Development & Review (5 hours)**
*Goal: Build out and refine the technical demonstrations for Talk 3.*
- **Task 5.1:** Finalize slides and build the data lineage/testing demo for Talk 3 (2.5 hours).
- **Task 5.2:** Perform a timed dry run of Talk 3 and adjust slide content based on timing (1.5 hours).
- **Task 5.3:** Create a list of recommended resources, packages, and books for attendees to continue learning about all three subjects (1 hour).

### **Week 6: Final Polish & Presenter Readiness (5 hours)**
*Goal: Perform final reviews, package the materials, and prepare for delivery.*
- **Task 6.1:** Conduct a final polish of all three slide decks, checking for consistent formatting, active links, and PyLadies branding (2 hours).
- **Task 6.2:** Package all presentation resources (slide PDFs, code sandboxes, feedback forms) into a public "Talks Resource Hub" folder (1.5 hours).
- **Task 6.3:** Write and schedule social media promotion templates for the kickoff announcements (1.5 hours).
